E-Commerce Customer Segmentation Report
Introduction
This project aims to segment the e-commerce website's customers to tailor marketing strategies effectively.
Methodology
  • Data Collection from Kaggle and UCI Repository
  • Data Preprocessing and Exploratory Data Analysis
  • Application of K-Means Clustering for Segmentation
Key Findings
Identified 4 distinct customer segments: Bargain Hunters, Loyal Customers, High-Value One-timers, and Window Shoppers.
Recommendations
  • Personalized email marketing for Loyal Customers
  • Special offers for Bargain Hunters to increase order value
  • Retargeting ads for Window Shoppers to convert them into buyers
